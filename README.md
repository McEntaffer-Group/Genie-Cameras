# Genie-Cameras
MATLAB and Python scripts for controlling DALSA Genie cameras and fitting output data


focuscamGUIfinal.mlapp is the main gui file
genieTest.m is a demonstration of how to read info from the genie. You install Matlab and the Image Processessing Toolbox and Support Package for GigE Vision Hardware. Then using admin privileges on the computer, you need to give Matlab network access.
